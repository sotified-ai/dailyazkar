# FILE 06 — 06-on-page-seo-optimization.md

### Scope

Optimize only **approved pages**.

### Allowed Optimizations

* Title tags
* Meta descriptions
* Internal links
* Schema (Article / FAQ)

### Forbidden

* Keyword stuffing
* Hidden text
* Auto‑generated backlinks

### Internal Linking Rule

* Link only to semantically related Azkar pages
* Max 5 links per page
